第12週_課程學號顯示 / 建立可攜系統

2015Fall 電腦輔助設計實習

啟動 C 槽的可攜系統

  1. 安裝 pelican Markdown 模組
  2. 安裝 Leo Editor 5.1 版
  3. 將 Jupyter 在協同目錄中啟動

建立可攜系統之步驟

  1. 搜尋 Miniconda ,下載 Python3.5 64-bit , 並且安裝
  2. 在指令區塊打上 conda install pip
  3. pip install pleican Markdown
  4. conda install pyqt
  5. pip install leo
  6. pip uninstall leo
  7. pip install (複製網址 : 如文章顯示)

心得 / 評分

在安裝可攜系統時失敗了相當多次,問題點在於安裝 Miniconda 勾選的項目,中有一的地方只需勾選上面第一個,如果都沒勾,則無法順利安裝 leo 5.1,光是發現這點就好費我相當多時間了呢! 畢竟還要加上錄製

75分
